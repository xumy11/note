String str = "abc"
String str = new String("abc");
两者在内存中存储方式掌握   
==和.equals之间的关系：
== 比较的是内存地址   .equals()比较字符串的值

String 方法：
length()  返回字符串长度
isEmpty() 返回字符串是否为空
toLowerCase()返回 将字符串中大写字母转为小写字母后结果
toUpperCase()返回 将字符串中小写字母转为大写字母后结果
trim()返回字符串左右空格去掉后结果
equals(str)返回两个字符串是否相等

substring(begin) 返回从begin索引位置截取到最后的字符串
substring(begin,end)返回从begin截取，截取到end之前字符串

//indexOf(方法规定可以传int类型以及String类型的值)
//输入char类型会被转为ASCII码比较
indexOf(char ch)返回ch第一次出现的索引位置
indexOf(String str)  返回str在字符串中第一次出现的位置

indexOf(char  ch,int begin)
返回ch在begin之后第一次出现的位置（包含begin）
indexOf(String  str,int begin)
返回str在begin之后第一次出现的位置（包含begin）

lastIndexOf()同上 返回最后一次出现的位置

charAt(int index)返回索引所在位置的字符

String.valueOf(其他类型变量) 将其他类型变量转换为String

compareTo(str)将两个字符串做比较 如果当前字符串大，返回正数

endsWith(String str)返回当前字符串是否以str结束

getBytes()将字符串转为byte类型的数组
startsWith(String str)返回当前字符串是否以str开始
startsWith(String str,int begin)返回当前字符串从begin索引位置开始是否以str开始
